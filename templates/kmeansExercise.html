Dataset

Age & Income Dataset

No labels or predefined categories are included, making this dataset appropriate for unsupervised learning. The two-dimensional structure allows full visual inspection of cluster formation across iterations.

Distance Calculation Method

Euclidean Distance

All assignments are based on the Euclidean distance between each data point and each centroid. For a point (Age, Income) and centroid (Cₓ, Cᵧ):

d = √( (Age − Cₓ)² + (Income − Cᵧ)² )

Each point is assigned to the cluster with the smallest distance. The same formula is applied to all 100 records in each iteration.

Interpretation of results

The total SSE decreased significantly across the three iterations — from 25,285,114 in iteration 1 to 15,411,803 in iteration 3, a total reduction of approximately 39%. This confirms that the algorithm progressively moved centroids toward more compact and representative positions.

The largest improvement occurred between iterations 1 and 2 (−33%), driven by the redistribution of boundary points between clusters. By iteration 3, the SSE curve began to flatten, indicating that the model was approaching convergence — a near-optimal clustering solution.

Final clustering result

After iteration 3, centroids showed minimal movement and cluster assignments were largely stable. The algorithm can be considered converged — running additional iterations would produce negligible improvements in SSE.
